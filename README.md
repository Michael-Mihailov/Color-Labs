# colorlabs21-Michael-Mihailov

A project working on image manipulation using Java. I used various image manipulation techniques and implemented my own Circle packing algorithm to proceduraly generate a cool poster. Final poster shown here.
![poster image](https://github.com/Michael-Mihailov/colorlabs21-Michael-Mihailov/blob/master/colorlabs21-Michael-Mihailov-20230521T021646Z-001/colorlabs21-Michael-Mihailov/ColorLabs17/images/Canvas_Final.png?raw=true)

Go to the readMe text file in the images folder for a detailed description of the image manipulations performed :)

This project uses existing libraries which I did not make and was created as a part of a open-ended school project for AP Computer Science at Naperville Central Highschool.
